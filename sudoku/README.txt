sudoku.py is programmed in python 2.7

To run this program from the command line, simply execute 'python sudoku.py' with python 2
The program should request a fileName, although a file name can optionally be passed as an argument to the script
Note that it is not required that the size of the board be given as the script deduces board size

Output will be shown on the console

To run an executable of this program, please run sudoku.exe, located in the dist folder
Note that this executable is unstable and may crash due to issues with z3 pathing
